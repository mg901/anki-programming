## [268 Missing Number 'arithmetic-sequence'](https://leetcode.com/problems/missing-number/description/)

<!-- notecardId: 1764093394748 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/WnPLSRLSANE

// - Time: O(n)
// - Space: O(1)
function missingNumber(nums) {
  const n = nums.length;
  const total = (n * (n + 1)) / 2; // (n + 1 * (n + 0)) / 2
  const sum = nums.reduce((acc, num) => acc + num, 0);

  return total - sum;
}
```
