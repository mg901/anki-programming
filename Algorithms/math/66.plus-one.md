## [66 Plus One](https://leetcode.com/problems/plus-one/description/)

<!-- notecardId: 1747590106365 -->

```js
function plusOne(digits) {
  for (let i = digits.length - 1; i >= 0; i -= 1) {
    digits[i] += 1;

    if (digits[i] < 10) return digits;
    digits[i] = 0;
  }

  digits.unshift(1);

  return digits;
}
```
