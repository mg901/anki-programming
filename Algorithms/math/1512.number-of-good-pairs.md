## [1512 Number of Good Pairs 'Math'](https://leetcode.com/problems/number-of-good-pairs/description/)

### Combination Formula

$$C(n, k) = \frac{n!}{k! \, (n - k)!}$$

### n Choose 2 Formula

$$C(n, 2) = \frac{n!}{2! \, (n - 2)!}$$

Expanding the factorials:

$$C(n, 2) = \frac{n \times (n - 1)}{2}$$

```js
// Explanation:
// - Neetcode: https://youtu.be/BqhDFUo1rjs

// Formula: (c * (c - 1)) / 2

// Find all possible combinations of equal
// numbers and remove mirrored duplicates (order doesnâ€™t matter)
//  by removing by 2

// - Time: O(n)
// - Space: O(n)
function numIdenticalPairs(nums) {
  const counter = new Map();
  let count = 0;

  for (const num of nums) {
   counter.set(num,  counter.get(num) ?? 0) + 1);
  }

  for (const freq of counter.values()) {
    count += Math.floor((freq * (freq - 1)) / 2);
  }

  return count;
}
```
