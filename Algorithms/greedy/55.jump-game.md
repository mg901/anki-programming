## [55 Jump Game](https://leetcode.com/problems/jump-game/description/)

<!-- notecardId: 1768484020418 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/Yan0cv2cLy8

// - Time: O(n)
// - Space: O(1)
function canJump(nums) {
  const n = nums.length;
  let goal = n - 1;

  for (let i = n - 2; i >= 0; i -= 1) {
    if (i + nums[i] >= goal) {
      goal = i;
    }
  }

  return goal === 0;
}
```
