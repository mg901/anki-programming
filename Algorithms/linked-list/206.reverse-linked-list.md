## [206 Reverse Linked List](https://leetcode.com/problems/reverse-linked-list/description/)

<!-- notecardId: 1741870233776 -->

```js
function reverseList(head) {
  if (!head) return null;

  let current = head;
  let prev = null;

  while (current) {
    const next = current.next;
    current.next = prev;
    prev = current;
    current = next;
  }

  return prev;
}
```
