## [2181 Merge Nodes in Between Zeroes](https://leetcode.com/problems/merge-nodes-in-between-zeros/description/)

<!-- notecardId: 1764776620758 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/jrSav7fulJY

// - Time: O(n)
// - Space: O(1)
function mergeNodes(head) {
  let current = head.next;

  while (current?.next) {
    if (current.next.val !== 0) {
      current.val += current.next.val;
      current.next = current.next.next;
    } else {
      current.next = current.next.next;
      current = current.next;
    }
  }

  return head.next;
}
```
