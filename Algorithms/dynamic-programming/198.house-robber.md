## [198 House Robber](https://leetcode.com/problems/house-robber/description/)

<!-- notecardId: 1752139596058 -->

```js
// Fibonacci numbers
function rob(nums) {
  let rob1 = 0;
  let rob2 = 0;

  for (const num of nums) {
    const temp = Math.max(rob1 + num, rob2);
    rob1 = rob2;
    rob2 = temp;
  }

  return rob2;
}
```
