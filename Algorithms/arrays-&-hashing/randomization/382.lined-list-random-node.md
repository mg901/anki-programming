## [382 Linked List Random Node](https://leetcode.com/problems/linked-list-random-node/description/)

<!-- notecardId: 1766866355358 -->

```js
class Solution {
  // - Time: O(n)
  // - Space: O(0)
  constructor(head) {
    this.arr = [];

    while (head) {
      this.arr.push(head.val);
      head = head.next;
    }
  }

  // - Time: O(1)
  getRandom() {
    const n = this.arr.length;
    const randomIdx = Math.floor(Math.random() * n);

    return this.arr[randomIdx];
  }
}
```
