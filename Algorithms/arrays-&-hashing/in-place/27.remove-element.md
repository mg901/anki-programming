## [27 Remove Element](https://leetcode.com/problems/remove-element/description/)

<!-- notecardId: 1764088529447 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/Pcd1ii9P9ZI

// Pattern:
// - Two Pointers

// Sub-Pattern: Slow / Fast
//  - Specific : Compression

// - Time: O(n)
// - Space: O(1)
function removeElement(nums, val) {
  const n = nums.length;
  let k = 0;

  for (let i = 0; i < n; i += 1) {
    if (nums[i] !== val) {
      nums[k] = nums[i];
      k += 1;
    }
  }

  return k;
}
```
