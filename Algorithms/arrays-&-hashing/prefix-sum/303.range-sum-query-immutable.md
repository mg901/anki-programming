## [303 Range Sum Query - Immutable](https://leetcode.com/problems/range-sum-query-immutable/description/)

```js
// Explanation:
// - Algorithm:
//    Profound Academy: https://youtu.be/xbYr9JOC2Lk

// - Problem:
//    Neetcode: https://youtu.be/2pndAmo_sMA

// - Time: O(n)
// - Space: O(1)
class NumArray {
  #prefix;

  constructor(nums) {
    const n = nums.length;
    this.#prefix = new Array(n + 1).fill(0);

    for (let i = 1; i <= n; i += 1) {
      this.#prefix[i] = this.#prefix[i - 1] + nums[i - 1];
    }
  }

  sumRange(left, right) {
    return this.#prefix[right + 1] - this.#prefix[left];
  }
}
```
