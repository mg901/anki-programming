## [226 Invert Binary Tree](https://leetcode.com/problems/invert-binary-tree/description/)

<!-- notecardId: 1744137642021 -->

```js
function invertTree(root) {
  if (!root) return null;

  const { left } = root;
  root.left = invertTree(root.right);
  root.right = invertTree(left);

  return root;
}
```
