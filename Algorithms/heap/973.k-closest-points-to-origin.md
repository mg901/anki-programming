## [973 K Closest Points to Origin](https://leetcode.com/problems/k-closest-points-to-origin/description/)

<!-- notecardId: 1767106755894 -->

The Euclidean distance formula between two points:

$$\sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$$

If (x2, y2) = (0, 0), then:

$$\sqrt{(x - 0)^2 + (y - 0)^2} = \sqrt{x^2 + y^2}$$

```js
// Explanation:
// - Neetcode: https://youtu.be/rI2EBUEMfTk

// - Time: O(n log(k))
// - Space: O(k)
function kClosest(points, k) {
  const maxpq = new MaxPriorityQueue(([distance]) => distance);

  for (const [x, y] of points) {
    const distance = x ** 2 + y ** 2;

    maxpq.enqueue([distance, x, y]);

    if (maxpq.size() > k) {
      maxpq.dequeue();
    }
  }

  return maxpq.toArray().map(([_, x, y]) => [x, y]);
}
```
