## [703 Kth Largest Element in a Stream](https://leetcode.com/problems/kth-largest-element-in-a-stream/description/)

<!-- notecardId: 1767098706988 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/hOjcdrqMoQ8

class KthLargest {
  // - Time: O(n log(k))
  // - Space: (k)
  constructor(k, nums) {
    this.k = k;
    this.maxpq = new MinPriorityQueue(null, nums);

    while (this.maxpq.size() > k) {
      this.maxpq.dequeue();
    }
  }

  add(val) {
    this.maxpq.enqueue(val);
    if (this.maxpq.size() > this.k) {
      this.maxpq.dequeue();
    }

    return this.maxpq.front();
  }
}
```
