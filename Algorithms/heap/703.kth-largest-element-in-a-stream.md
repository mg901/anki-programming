## [703 Kth Largest Element in a Stream](https://leetcode.com/problems/kth-largest-element-in-a-stream/description/)

<!-- notecardId: 1767478608066 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/hOjcdrqMoQ8

class KthLargest {
  // - Time: O(n log(k))
  // - Space: (k)
  constructor(k, nums) {
    this.k = k;
    this.minHeap = new MinHeap(null, nums);

    while (this.minHeap.size() > k) {
      this.minHeap.pop();
    }
  }

  add(val) {
    this.minHeap.push(val);
    if (this.minHeap.size() > this.k) {
      this.minHeap.pop();
    }

    return this.minHeap.top();
  }
}
```
