## [190 Reverse Bits](https://leetcode.com/problems/reverse-bits/description/)

<!-- notecardId: 1765373532112 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/UcoN6UjAI64

// - Time: O(n)
// - Space: O(1)
function reverseBits(n) {
  let reversed = 0;

  for (let i = 0; i < 32; i += 1) {
    reversed = (reversed << 1) | (n & 1);
    n >>= 1;
  }

  return reversed;
}
```
