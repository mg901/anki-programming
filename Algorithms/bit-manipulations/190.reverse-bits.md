## [190 Reverse Bits](https://leetcode.com/problems/reverse-bits/description/?envType=study-plan-v2&envId=top-interview-150)

<!-- notecardId: 1740477906826 -->

```js
function reverseBits(n) {
  let acc = 0;

  for (let i = 0; i < 32; i += 1) {
    acc = (acc << 1) | (n & 1);
    n >>= 1;
  }

  return acc >>> 0;
}
```
