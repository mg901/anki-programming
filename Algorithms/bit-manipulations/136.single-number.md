## [136 Single Number](https://leetcode.com/problems/single-number/description/?envType=study-plan-v2&envId=top-interview-150)

<!-- notecardId: 1740478867870 -->

```js
function singleNumber(nums) {
  let unique = 0;

  for (let i = 0; i < nums.length; i += 1) {
    unique ^= nums[i];
  }

  return unique;
}
```
