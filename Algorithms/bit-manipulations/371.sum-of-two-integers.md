## [371 Sum of Two Integers](https://leetcode.com/problems/sum-of-two-integers/description/)

<!-- notecardId: 1761920483824 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/gVUrDV4tZfY

// - Time: O(1)
// - Space: O(1)
function getSum(a, b) {
  while (b !== 0) {
    let carry = (a & b) << 1;

    a ^= b;
    b = carry;
  }

  return a;
}
```
