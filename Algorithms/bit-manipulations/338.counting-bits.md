## [338 Counting Bits](https://leetcode.com/problems/counting-bits/description/)

<!-- notecardId: 1740487231991 -->

```js
const countBits = function (n) {
  const results = new Array(n + 1).fill(0);

  for (let i = 1; i <= n; i += 1) {
    results[i] = results[i >> 1] + (i & 1);
  }

  return results;
};
```
