## [338 Counting Bits](https://leetcode.com/problems/counting-bits/description/)

<!-- notecardId: 1761840529561 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/RyBM56RIWrM

// - Time: O(n)
// - Space: O(n)
function countBits(n) {
  const dp = new Uint8Array(n + 1);

  for (let i = 1; i <= n; i += 1) {
    dp[i] = dp[i >> 1] + (i & 1);
  }

  return dp;
}
```
