## [231 Power of Two](https://leetcode.com/problems/power-of-two/description/)

<!-- notecardId: 1768569051718 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/H2bjttEV4Vc

// Extract least significant bit: n & -n
// - Option 1 - value equals its lowest set bit
//    (n & -n) === n

// - Option 2 - clear LSB via XOR
//    ((n & -n) ^ n) === 0

// - Time: O(1)
// - Space: O(1)
function isPowerOfTwo(n) {
  return n > 0 && (n & -n) === n;
}
```

## [231 Power of Two 'Brian-Kernighan'](https://leetcode.com/problems/power-of-two/description/)

<!-- notecardId: 1768568359890 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/H2bjttEV4Vc

// - Algorithm: Brian-Kernighan

// Remove least significant bit:
// - Option 1 — canonical bit invariant
//    (n & (n - 1)) === 0

// - Option 2 — subtract lowest set bit (LSB)
//    (n - (n & -n)) === 0

// - Option 3 — explicit two’s complement form
//    (n - (n & (~n + 1))) === 0

// - Time: O(1)
// - Space: O(1)
function isPowerOfTwo(n) {
  return n > 0 && (n & (n - 1)) === 0;
}
```
