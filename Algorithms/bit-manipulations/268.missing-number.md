## [268 Missing Number](https://leetcode.com/problems/missing-number/description/)

<!-- notecardId: 1761852605151 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/WnPLSRLSANE

// - Time: O(n)
// - Space: O(1)
function missingNumber(nums) {
  const n = nums.length;
  let missing = 0;
  let i = 0;

  while (i < n) {
    missing ^= nums[i] ^ i;
    i += 1;
  }

  return missing ^ i;
}
```
