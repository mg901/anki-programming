## [268 Missing Number 'xor'](https://leetcode.com/problems/missing-number/description/)

<!-- notecardId: 1762779377050 -->

```js
// Explanation:
// - Neetcode: https://youtu.be/WnPLSRLSANE

// - Time: O(n)
// - Space: O(1)
function missingNumber(nums) {
  const n = nums.length;
  let xor = n;

  for (let i = 0; i < n; i += 1) {
    xor ^= nums[i] ^ i;
  }

  return xor;
}
```
